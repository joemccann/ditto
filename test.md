# Markdown to PDF Test Document

This is a test document to demonstrate the markdown to PDF converter capabilities.

## Features Supported

### Basic Text Formatting

This document tests **bold text**, *italic text*, and ***bold italic*** text.
It also tests ~~strikethrough~~ text.

### Lists

Unordered list:
- Item 1
- Item 2
  - Nested item 2.1
  - Nested item 2.2
- Item 3

Ordered list:
1. First item
2. Second item
3. Third item

Task lists:
- [ ] Incomplete task
- [x] Completed task
- [ ] Another task

### Blockquote

> This is a blockquote.
> It can span multiple lines.
